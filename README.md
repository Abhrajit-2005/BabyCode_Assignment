## Installation
# Clone the repository
```bash
git clone https://github.com/Abhrajit-2005/BabyCode_Assignment.git
cd my-project
```
# Install dependencies
```bash
npm install
#or
yarn install
```
# Start the development server
```bash
npm start
#or
yarn start
```
# Open your browser
Navigate to http://localhost:5173 to view the application.
